# api

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Setup environment variables:

```bash
#.env

EDGEDB_AUTH_BASE_URL=
PORT=
BASE_URL=
```

This project was created using `bun init` in bun v1.1.6. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
